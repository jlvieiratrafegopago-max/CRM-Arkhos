# 🏛️ CRM Agência Arkhos

Sistema de Gerenciamento de Relacionamento com Cliente (CRM) desenvolvido para a **Agência Arkhos**, focado em gestão de contratos de marketing digital, análise de dados e serviços de tecnologia.

Este projeto faz parte do meu portfólio acadêmico no curso de **Inteligência Artificial na Uninassau**.

## 🚀 Funcionalidades

- **Acesso Restrito:** Painel administrativo protegido por senha para gestão de dados.
- **Dashboard em Tempo Real:** Visualização de métricas como faturamento total, ticket médio e número de contratos ativos.
- **Gestão de Clientes (CRUD):** - Cadastro de novos clientes e serviços.
  - Edição completa de registros existentes (nome, valor, status, serviço, etc.).
  - Exclusão de registros da base de dados.
- **Persistência de Dados:** Integração com banco de dados **SQLite** para armazenamento local e em nuvem.
- **Interface Moderna:** Desenvolvido com **Streamlit** para uma experiência de usuário fluida e responsiva.

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** Python 3.x
- **Framework Web:** [Streamlit](https://streamlit.io/)
- **Banco de Dados:** SQLite3
- **Análise de Dados:** Pandas
- **Hospedagem:** Streamlit Cloud
